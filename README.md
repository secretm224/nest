## 카펜스트리트 테스트 전형 

# 환경: 

* nestjs (swagger/typeorm/httpmodule)
* Mysql 8.0 ,Mysql Workbench

# API 정보
** 과제 설명 링크 된 번역 참고 API 링크가 접근이 안되어 파파코 api 신규 앱 등록하여 개발 진행 하였습니다. **
* api url (http://localhost:3000/api)
